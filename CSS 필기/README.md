# CSS

## 정의 의미
>Cascading Style Sheet

1. Author style
>우리가 작성한 스타일 시트 (웹사이트를 만들고 제공하는 CSS파일)

2. User style
>브라우저에서 사용자가 보기 쉽게 사용자가 지정한 시트

3. Browser
>브라우저에서 기본적으로 지정된 스타일 시트

## Selectors
>HTML의 어떤 태그들을 고를지 결정하는 것

```
*: 모든 태그들을 고르는 것
tag이름: 그 태그를 고르는 것
ID: 해당하는 ID만 고르는 것
class: 해당하는 class만 고르는 것
```

## CSS 기본문법

```
selector {
  property: value;
}
```
